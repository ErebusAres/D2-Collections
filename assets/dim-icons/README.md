Selected SVGs from justrealmilk/destiny-icons, the DIM submodule used for Destiny icon glyphs.

Source commit: 394ed051455e938f72ddd600d42cf87600ec7172
License: CC0 1.0 Universal

Armor slot icons come from `armor_types/`.
Class icons are Bungie-hosted Destiny 2 class emblems vendored from `www.bungie.net/common/destiny2_content/icons/`.
`dim_*` icons are small UI glyphs sourced from Destiny Item Manager:
- Font Awesome glyphs from DIM's `src/app/shell/icons/Library.js` / subset webfont.
- Custom DIM glyphs from `src/app/shell/icons/custom/`.
- Progress/engram/tracked SVGs from DIM's `src/images/`.
